# 🏎️ FaceRacer - Kafa Kontrol Oyunu

**Copyright © 2026 Hakan Çetin - Tüm hakları saklıdır.**

FaceRacer, yüz takibi teknolojisi kullanarak kafa hareketleriyle oynanan yeni nesil bir 3D araba yarışı oyunudur. Engelli bireyler ve ofis çalışanları için erişilebilir bir oyun deneyimi sunar.

## 🎯 Özellikler

- **Yüz Takibi Kontrolü**: MediaPipe Face Mesh ile gerçek zamanlı yüz algılama
- **Küçük Hareket Duyarlılığı**: Alın hareketleriyle hassas kontrol
- **3D Grafikler**: Three.js ile immersive oyun deneyimi
- **Erişilebilirlik**: Klavye veya mouse gerektirmez
- **Nitro Sistemi**: Göz kırpma ile hız artışı
- **Otomatik Kalibrasyon**: 5 saniyelik kişisel ayar

## 🎮 Nasıl Oynanır

1. **Kalibrasyon**: Yüzünüzü çerçevenin tam ortasına getirin (alnınız merkezde olsun)
2. **Dönüş Kontrolü**: Alnınızı sağa/sola hafifçe kaydırın
3. **Hız Kontrolü**: Alnınızı yukarı/aşağı hafifçe kaydırın
4. **Nitro**: Göz kırpın

## 🛠️ Teknoloji Stack

- **Three.js**: 3D grafik motoru
- **MediaPipe Face Mesh**: Yüz landmark tespiti
- **WebRTC**: Kamera erişimi
- **HTML5/CSS3/JavaScript**: Modern web teknolojileri
