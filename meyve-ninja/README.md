# Meyve Ninja 🍉

El hareketleriyle kontrol edilen meyve kesme oyunu. Web kamerası aracılığıyla el hareketlerinizi takip eder ve işaret parmağınızla meyveleri kesersiniz.

## Özellikler

- 🖐️ **El Takibi**: MediaPipe ile gerçek zamanlı el takibi
- 🍎 **5 Farklı Meyve**: Karpuz, Portakal, Elma, Limon, Üzüm
- ⚡ **Combo Sistemi**: Ardışık kesmelerle daha yüksek puan
- 💫 **Görsel Efektler**: Kesme izleri, patlama animasyonları
- ❤️ **3 Can**: Kaçırdığınız meyvelerde can kaybedersiniz

## Gereksinimler

- Python 3.8+
- Web kamerası

## Kurulum

```bash
pip install -r requirements.txt
```

## Çalıştırma

```bash
python meyve_ninja.py
```

## Nasıl Oynanır

1. Web kameranızın önünde durun
2. İşaret parmağınızı kaldırın
3. Parmağınızı hızlıca hareket ettirerek meyveleri kesin
4. Meyveleri kaçırmayın!

## Kontroller

- **R**: Oyun bittiğinde tekrar oyna
- **ESC**: Çıkış

## Teknolojiler

- OpenCV - Kamera görüntüsü
- MediaPipe - El takibi
- Pygame - Oyun grafikleri
