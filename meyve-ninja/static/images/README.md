# Meyve Görselleri

Bu klasöre gerçek meyve görsellerini koyun. Şeffaf arka planlı PNG dosyaları kullanın.

## Gerekli Görseller:

1. **karpuz.png** - Karpuz (yaklaşık 128x128px)
2. **portakal.png** - Portakal (yaklaşık 100x100px)
3. **elma.png** - Elma (yaklaşık 110x110px)
4. **limon.png** - Limon (yaklaşık 90x90px)
5. **uzum.png** - Üzüm (yaklaşık 80x80px)
6. **muz.png** - Muz (yaklaşık 120x120px)
7. **bomba.png** - Bomba (yaklaşık 110x110px)

## Görsel Özellikleri:
- Şeffaf arka planlı PNG
- Merkezde meyve
- Yeterli çözünürlük (en az 100x100px)
- Gerçekçi görünüm

## Görsel Bulabileceğiniz Kaynaklar:
- Google Images (transparent PNG ile ara)
- Freepik, Pixabay (ücretsiz stok görseller)
- OpenGameArt (oyun görselleri)

Görselleri bu klasöre koyduktan sonra tarayıcıyı yenileyin (F5).
